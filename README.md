### 项目依赖
- vue-router的hash模式控制路由
- vuex控制用户登录登出，token存储等
- axios发起后台，进行相应拦截
- scss进行css预编译
- es6开发模式
- antD作为UI框架
- echarts作为图表框架
- iconfont作为字体图标库
